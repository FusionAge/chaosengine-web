Discord Bot for Diablo:Immortal Clans

Features:
  • Uses Discord-recommended slash commands
  • Simple setup (just use /start and fill in Clan Name & Battle.net Server)
  • Users add themselves via /register with their Class & Character Name
  • Store Paragon Level, Combat Rating, Damage, Life, and Resonance
  • Easy-to-use tool to update info via webpage (use /tool)
  • Manage Set Items (equipped and inventory)
  • "Wanted Sets" to help you complete your set pieces
  • View your clan members & their stats/progress
  • Lookup set information easily (use /setinfo)
  • ...to get started, click the big-ass button below!

Installation
  1. Click the "INVITE BOT TO DISCORD" button on the homepage
  2. The Bot "ChaosEngine" should appear in your channel
  3. Type /start to begin (it will assign you as an admin)
  4. That's it! ChaosEngine is ready to use
Recommended Settings
  1. By default, ChaosEngine listens on every channel it has permission to. This could create quite a bit of noise, so it is recommended that you assign ChaosEngine to its own channel (like #CHAOSENGINE) so that your users don't invoke it in every channel and ChaosEngine pollutes the channel with responses.
    To limit ChaosEngine to a specific channel:
    1. Right-click on ChaosEngine in the Members List
    2. Select "Manage Integrations"
    3. In the Channels section, Disable "All Channels" (red X")
    4. Click the "Add Channels" button
    5. Check the box next to the channel you want ChaosEngine to listen to
    6. Click the Add Button
    7. Click "Save Changes"

Set Items Tool
How to access the tool

Note: It is recommended to use a desktop browser for the best experience.

  • Use the /tool command in discord
  • You will receive a private message from ChaosEngine with a link to a web page
  • IMPORTANT! Do not share your link or someone else can change (or delete) your data
  • When clicking the link you will first see your stats, which is editable
  • A tab/link for "My Sets" is where the Set Item tool is found

There are three sections to the "My Sets" feature: Wanted Sets, Equipped Items, and Inventory Items.

"Wanted Sets"

This is the list of sets you are looking to obtain. To add a Set you want, select a set from the drop-down and add it.

ChaosEngine will track your set items against this set, telling you how many items you have belonging to this set IN TOTAL as well as COMPLETION (having at least one set item each slot).

You can also see what dungeon(s) to farm in order to get the set items needed to complete the set. When the item drops for you, add it to your inventory and/or equipped items.

Clicking the information icon will tell you about the set bonuses.

"Equipped Items"

This is the list of set items you have equipped. In the "Set" column, the green number indicates the number of items from that set you have equipped. To add an item from your inventory to this set, see the "Inventory Items" below.

"Inventory Items"

Here is where you add your set items, as they drop, as well as replace items in your Equipped items.

Click the Plus button to add a new item to your inventory. Select the set, and the slot, then enter the values for Intelligence/Strength, Vitality, Willpower, Fortitude and click "Add"

Click the Checkmark button to add additional filters.

Click the Question mark button to show a help dialog

To add an item from inventory to equipped items, click the square-plus button to the left of the item. This will replace the item for that slot in the Equipped items, and move the old equipped item back to inventory.

To delete an item from inventory, click the trashcan icon to the left of the item

You can search and sort the list of inventory items. It searches ALL fields, and yes, you can multi-sort. Hold down shift while clicking on the up/down arrows in the header column. (I have no idea how to do this on mobile, sorry)
